# AI YouTube Automation

This project automates the process of creating and uploading YouTube videos using n8n and AI tools.

## What this project does
- Takes a user idea from a form
- Generates a video prompt using AI
- Generates a title and hashtags using AI
- Generates a video using an external AI service
- Uploads the video automatically to YouTube

## Tools used
- n8n
- AI agents
- Webhooks
- YouTube API

## Status
This project is a work in progress and is being improved step by step.
